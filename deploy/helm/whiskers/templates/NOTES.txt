Whiskers ist installiert.

1. Auf den Pod warten:
   kubectl -n {{ .Release.Namespace }} rollout status deploy/{{ include "whiskers.fullname" . }}

2. UI öffnen (ohne Ingress):
   kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "whiskers.fullname" . }} 8080:{{ .Values.service.port }}
   → http://localhost:8080  (Erststart: Setup-Wizard legt den Admin an)
